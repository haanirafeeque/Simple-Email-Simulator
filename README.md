# Python Email Management System

A simple console-based email management system built using Python and Object-Oriented Programming (OOP).

## Features
- Send emails
- Receive emails
- View inbox
- Read emails
- Delete emails
- Automatic read status
- Email timestamps

## Technologies
- Python 3
- Object-Oriented Programming (OOP)
- `datetime` module

## Run

```bash
python main.py
```
